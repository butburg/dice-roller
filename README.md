# Dice Roller

This repository contains a small interactive Dice Roller component built with TSX.

- Roll between **1 and 4 dice**.
- All dice can have a different number of faces (for example: d4, d6, d8, d10, d12, d20).
- The UI lets you select how many dice to roll and the face-count for the dice, then shows individual results, the total a history.
- slider for some fun color change

## Run locally

- Install dependencies: `npm install`
- Start dev server: `npm run dev`

## Template

This repository was created from the TSX template: https://github.com/butburg/tsx-template
